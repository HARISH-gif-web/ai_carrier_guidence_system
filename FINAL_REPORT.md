## 🚀 HOW TO USE

### Quick Start (30 seconds)

**Windows:**
```bash
start-dev.bat
```

**Linux/Mac:**
```bash
bash start-dev.sh
```

**Docker:**
```bash
docker-compose up --build
```

### Manual Start

```bash
# Install dependencies
cd backend
pip install -r requirements.txt

# Start backend
uvicorn main:app --reload

# In another terminal, open index.html in browser
```
