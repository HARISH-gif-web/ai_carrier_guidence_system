⚡ Quick Start

1. Windows Users

Run the batch script

start-dev.bat

2. Linux/Mac Users

Run the shell script

bash start-dev.sh

3. Docker Users

Run with Docker Compose

docker-compose up --build

4. Manual Start

Run backend manually

cd backend
uvicorn main:app --reload
→ Go to Application
